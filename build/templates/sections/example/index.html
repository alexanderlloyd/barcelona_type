Ahoi, bootstrap-protos!

this is powered by the jade template engine and served via node.js

until now this is what you got. Only jQuery is included via bower. No other fancy js and css frameworks.

but you can add them very easily. Just update your bower file and run bower install

All jade files are supposed to be located under app/jade/. All .jade files with an _underscore are treated as partials and not rendered directly. Everything else is a jade file that is supposed to be a page.

And this is how i thought about the directory structure of the jade directory

jade
|-- index.jade
|-- layouts
|   |-- _default.jade
|-- page1
|  |-- index.jade
|       |-- partials
|           |-- _partial.jade
+ - page2
   |-- index.jade
        |-- partials
            |-- _partial.jade
